Consider the following code:

interface FuncInter1 { 
        int operation(int a, int b); 
    } 

interface FuncInter2 { 
        void sayMessage(String message); 
    } 
   
public class Test { 
	 
    private int operate(int a, int b, FuncInter1 fobj) { 
	    return fobj.operation(a, b); 
	} 

    public static void main(String args[]) { 
       
        FuncInter1 add = (int x, int y) -> x * y; 
        FuncInter1 multiply = (int x, int y) -> x + y; 
        Test tobj = new Test(); 
        System.out.println("Addition is " + tobj.operate(6, 3,add)); 
        System.out.println("Multiplication is " + tobj.operate(6, 3,multiply)); 
  
        FuncInter2 fobj = message ->System.out.println("Hello "+ message); 
        fobj.sayMessage("Infosys"); 
    } 
}

What will be the output?

Unknown Exception will be thrown.
Invalid interface usage Exception will be thrown as we cannot use same functional interface for more than 1 purpose at one time
Addition is 18 Multiplication is 9 Hello Infosys
